 Давайте разберемся, что такое переменные 
          на простом примере.


  Создайте переменную teacher и 
        присвойте ей значение c именем вашего преподавателя.


teacher = "Имя"
print(teacher)
>>> Имя 

    
  Слово Имя на забудьте заменить на настоящее имя!

  Чтобы сохранить строку, нужно заключить текст в кавычки. 
  В Python вы можете использовать для этой цели 
  как одинарные, так и двойные кавычки. Допустимы оба варианта, 
  но кавычки в начале и в конце строки должны быть одного типа.

    Например:


 teacher = "Имя"


Это, конечно, мелочи. Но дьявол кроется в деталях :)

  Теперь мы можем обращаться к переменной и выполнять с ней операции. 

  Например, выведите имя учителя 20 раз:
    

 print(teacher * 20) 


   Имена можно присваивать так же числовым значениям:


  A = 5   
  B = 6   
  A + B   


 Также переменную можно приравнять саму к себе.
  Это все равно что утверждать: «Я — это я»:

  
  klass = 8
  klass = klass 
     
  print(klass)
  >>> 8

  
  Однако небольшое изменение может сделать вас совершенно новой сущностью:

  
  klass = klass + 1
    print(variable)
    >>> 9


  Или:
  
  kklass = 8
    klass += 1
    print(klass )
    >>> 9


    
     При работе с переменными, 
  важно  учитывать несколько правил:
    

  1. Можно использовать только латинские буквы, цифры и нижние подчеркивания;
    2. 

  2. Имя переменной не может начинаться с числа;
    3. 

  3. Чувствительно к регистру, те Teacher и teacher – это две разные переменные;
    4. 

  4. Вам будет гораздо удобнее работать с кодом, 
если название переменных будет отражать суть 
  (например, не lkjn, a myTeacher).
